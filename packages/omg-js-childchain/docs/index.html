ChildChain

Summary: Interact with Tesuji Plasma Childchain from JavaScript (Node.js and Browser) Description: allows user to interact with Tesuji Plasma from JavaScript. look up examples for implementations in boith Client and Server

new ChildChain(watcherUrl: string, childChainUrl: string)
Parameters
watcherUrl (string) contains the url of the watcher server
childChainUrl (string) contains the url of the childchain server to communicate with
