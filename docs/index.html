Omg

Interact with Tesuji Plasma Childchain & Rootchain from JavaScript (Node.js and Browser)

new Omg(watcherUrl: string, childChainUrl: string, web3Provider: string): object
Parameters
watcherUrl (string) contains the url of the watcher server
childChainUrl (string) contains the url of the childchain server to communicate with
web3Provider (string) contains the url of the geth node/ web3 provider
Returns
object: Omg (Omg.childchain and Omg.rootChain)
